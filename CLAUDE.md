# BUG-SQUASHER – Project Instructions

## Git Workflow (MANDATORY)

- At the start of every session: run `git log --oneline -10` and `git status` to orient before any work.
- After every completed step: immediately commit and push (`git add -A`, short clear message, `git push`).
- Push target is ALWAYS `origin` = `https://github.com/xyloxs/BUG-SQUASHER.git`. Never change this.
- Never push to any other remote. Verify with `git remote -v` if unsure.

## Commit Frequency

Commit after every meaningful unit of progress — not on a timer, but tied to completion:
- New file created
- Feature implemented
- Bug fixed
- Asset added
- Config changed
